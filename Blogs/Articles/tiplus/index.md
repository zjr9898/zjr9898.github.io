# 知几学生越狱教程

~知几学生吓哭了~

## 启动 adb

打开天学网 - 设置 - 个人信息收集清单 - 权限 - 位置信息 - 查看具有此权限的所有应用 - 右上角三个点 - 显示系统应用 - `Google Play` 服务 - 位置信息设置  - 应用的位置信息使用权 - 右上角三个点 - 显示系统应用 - 知几学生 - 查看“知几学生”的所有权限 - 知几学生的图标 - 高级 - 主屏幕应用 - 右上角搜索`ip ` 。


这时候你应当打开了墨水屏的设置。


划到最下面，点开关于，重复点击版本号直到打开开发人员设置，返回设置主页。


在上方搜索栏搜索 `ip`，打开第二个搜索结果。


这时候应当进入了无线调试的页面，在右上角可以打开并允许无线调试。


此时你已经成功启动 `adb` 了。

## 使用 adb 进行调试

前置知识：`adb `（这里不做赘述，不会的话上网查一下）

使用电脑或`termux`（手机上的开源`linux`终端）上的 adb 命令连接 `pad` 。

使用 `adb` 解锁安装未知应用权限。

方式如下：

```
adb shell
```

```
service call onyx_device_service 4 s16 APP_INSTALL_WHITE_LIST s16 '{"enable":false,"whiteList":[]}'
```

现在就可以安装自己的应用了，可以使用插在`pad`上的 `u` 盘上的 `apk`进行安装，也可以用`adb`安装手机或电脑上的`apk`。

使用上述方法安装`Nova Launcher`，将主屏应用改为`Nova Launcher`。

现在已经成功越狱，但是无法上网。

## 解决上网问题

注意到如果将知几学生清除数据或卸载更新，就可以上网。

但是在理想状态下我们应当同时保留知几学生的功能。

以下是解决方案。

打开设置的搜索栏搜`多用户`，添加一个用户`访客`。

将`机主`的知己学生清除数据（不能卸载更新），在访客页面
