## 启动 adb

打开天学网 - 设置 - 个人信息收集清单 - 权限 - 位置信息 - 查看具有此权限的所有应用 - 右上角三个点 - 显示系统应用 - Google Play 服务 - 位置信息设置  - 应用的位置信息使用权 - 右上角三个点 - 显示系统应用 - 知几学生 - 查看“知几学生”的所有权限 - 知几学生的图标 - 高级 - 主屏幕应用 - 右上角搜索`ip ` 。


这时候你应当打开了墨水屏的设置。


划到最下面，点开关于，重复点击版本号直到打开开发人员设置，返回设置主页。


在上方搜索栏搜索 ip，打开第二个搜索结果。


这时候应当进入了无线调试的页面，在右上角可以打开并允许无线调试。


此时你已经成功启动 adb 了。

## 使用 adb 进行调试

前置知识：adb （这里不做赘述，不会的话上网查一下）

使用电脑或termux（手机上的开源linux终端）上的 adb 命令连接 pad 。

使用 adb 解锁安装未知应用权限。

方式如下：

```
adb shell
```
